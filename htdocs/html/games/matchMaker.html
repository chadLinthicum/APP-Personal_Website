Match Maker

Match Maker is an Extension Exercise that will be added to the I.T.E. in 2024 and is currently in development.

The game is an edutainment title that challenges players to match labels with corresponding images in a given time. The aesthetic is inspired by yesterday's point and click strategy games including Fall Out. It consists of 40 images displayed on the screen, with five labels placed below them. Players can freely choose any label and attempt to match it with one of the images. If the match is correct, the image disappears. Otherwise, the label returns to its original position.
