Virtualize This!

Virtualize This! is an Extension Exercise that will be added to the I.T.E. in 2024 and is currently in development. The game is an edutainment title that teaches players about virtualization through memory games. Players match pairs of virtualization-related objects while exploring different levels of difficulty that introduce concepts like virtual machines, hypervisors, and containers. Virtualize This! aims to make virtualization accessible and enjoyable, providing an immersive way to learn about this technology through interactive gameplay.
