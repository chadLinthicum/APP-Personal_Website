I.T.E. - Integrated Training Environment

The Immersive Training Environment is a 3D environment created for the Marine Corps that provides the user with readily accessible training and troubleshooting instruction. The ITE comes pre-installed on workstations provided to CTP Operators in the Marine Corps. When launched, the ITE runs in the browser utilizing a Unity WebGL based 3D environment that allows the user to interact with the environment and launch an HTML5 based menu system to view lessons, checklists, quick reference videos, assessments and documentation. Additionally the ITE has a C# based local web server component that is bundled with the application that serves the purpose of authenticating users, serving content, and tracking user statistics throughout their use of the ITE.
